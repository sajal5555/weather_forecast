This application shows the weather data which include location, time, description, wind, pressure, humidity and temperature. There are following features:

1. Top Cities: This list has some predefined top Indian cities.

2. Map Locations: This list will display the location which we will select from map.

3. Bookmark: This list will display bookmarked location selected from `Top cities` or `Map locations`.

  • On clicking on the any list item, the weather forecast detail screen will display.

  • On clicking star icon in any list, location named on the row will be stored in bookmarked list.

  • On clicking cancel icon in any list, location from that list will be removed.

  • On Clinking menu button to open setting or menu screen. Through setting screen you can clean the stored locations and bookmarks. There you can change the temperature unit system also.
